📌-Sistema de Quiz com Gerenciador de Perguntas

Sobre o projeto

Este projeto consiste em um sistema completo de quiz interativo, desenvolvido em Python utilizando a biblioteca Tkinter para interface gráfica.

O sistema é dividido em duas partes principais:

🎮 Aplicação de Quiz (jogo)
🛠️ Gerenciador de perguntas (CRUD)

Isso permite não apenas jogar, mas também criar, editar e organizar perguntas dinamicamente, tornando o projeto totalmente personalizável.

🎮 Funcionalidades do Quiz

👥 Modo multiplayer (2 jogadores)<br>
🎯 Sistema de turnos alternados<br>
🔢 Escolha do número de rodadas<br>
📚 Seleção de temas (ex: gerais, cinema, esportes, etc.)<br>
🧩 Perguntas aleatórias a cada partida<br>
🏆 Sistema de pontuação em tempo real<br>
📊 Tela final com resultado (vitória ou empate)<br>
🔊 Efeitos sonoros (resposta correta, erro, cliques, etc.)<br>
🎨 Alternância entre tema claro e escuro<br>
🖥️ Interface gráfica totalmente interativa<br>

🛠️ Funcionalidades do Gerenciador (CRUD)

➕ Adicionar novas perguntas<br>
✏️ Editar perguntas existentes<br>
❌ Excluir perguntas<br>
🔄 Atualização em tempo real<br>
🗂️ Organização por temas<br>
💾 Armazenamento em arquivo JSON<br>
📋 Visualização em tabela (Treeview)<br>

🎨 Interface e UX

Sistema de temas (claro 🌞 e escuro 🌙)<br>
Interface moderna com cores personalizadas<br>
Feedback visual para ações do usuário<br>
Navegação fluida entre telas<br>
Componentes organizados e responsivos<br>

🧱 Tecnologias utilizadas

Python 🐍<br>
Tkinter (GUI)<br>
ttk (componentes avançados)<br>
JSON (armazenamento de dados)<br>
Pygame (efeitos sonoros)<br>

Objetivo<br>
Este projeto foi desenvolvido com foco em praticar lógica de programação, manipulação de dados e criação de interfaces gráficas, simulando um sistema real completo.
